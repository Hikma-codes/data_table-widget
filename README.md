Flutter DataTable Widget Demo

Overview

I have made a Flutter demo for the DataTable widget that I showed in class on February 12 2026. This DataTable widget is really cool. The DataTable widget can be used to show data in my app.

The DataTable widget is used to display data about stress percentages of students throughout the week. My app is an example of how to use the DataTable widget to show this kind of data. The DataTable widget is very useful, for this purpose. I have a demo made with Flutter. It has two tables that I use with the DataTable widget. The first table for the DataTable widget is about weeks 1 to 6. This table shows my stress levels. The other table for the DataTable widget is about weeks 7 to 11. This table for the DataTable widget shows what my stress levels might be like later, in the semester.

The main idea of this project was to show people how the DataTable thing works in Flutter. The DataTable is used to make tables that're easy to read. We have a table where each student gets their line and each week is a separate column. I made the cells in the DataTable colors to show how stressed the students are.
